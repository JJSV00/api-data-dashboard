# API Dashboard Project

This project is a learning-focused software development project where I build a cloud-ready application that integrates multiple APIs and displays data in a simple dashboard.

---

## 🚀 Project Goal

The goal of this project is to:
- Learn how to work with REST APIs
- Build a simple data dashboard
- Practice GitHub-based project management
- Understand cloud deployment using Microsoft Azure

---

## 🧩 Features

- API data fetching and integration
- Data processing and formatting
- Simple dashboard UI (planned/ongoing)
- Iterative development using MVP approach
- Documented development process (project journal)

---

## 🛠️ Tech Stack

- Python
- REST APIs
- Streamlit (or planned UI framework)
- Git & GitHub
- Microsoft Azure (for deployment)
